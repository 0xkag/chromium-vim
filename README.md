chromium-vim
============

#Keybindings

##Movement

| j, s | scroll down           (scrollDown)     |
| k, w | scroll up             (scrollUp)       |
| d    | scroll half-page down (scrollPageDown) |
| u, e | scroll half-page up   (scrollPageUp)   |
| l    | scroll right          (scrollRight)    |
| h    | scroll left           (scrollLeft)     |
