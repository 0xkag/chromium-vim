Command Mode (open with ":")
:tabopen <SEARCH/AUTOCOMPLETE>
Open a new tab with the typed/completed Url or Google search
:open <SEARCH/AUTOCOMPLETE>
Open the typed/completed Url or Google search in the same tab
:history <SEARCH/AUTOCOMPLETE>
Search through browsing history
:bookmarks <SEARCH/AUTOCOMPLETE>
Search through your bookmarks
:set <SEARCH/AUTOCOMPLETE>
Temporarily map a setting
:chrome:// <SEARCH/AUTOCOMPLETE>
Open one of the chrome://* Urls
:tabclose
Close the current tab
:nohl
Clears the page highlighting from the last search
:duplicate
Duplicate the current tab
:settings
Opens this settings page
